## Getting Started

### Prerequisites
- [Node](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)
  
### Installation
Clone the repro
```shell
git clone git@bitbucket.org:sala7design/trular.git
```

Install dependences
```shell
yarn install
```

To run in development mode
```shell
yarn start
```

### Production
To run in production mode
```shell
yarn build
```

### Version
Create path
```shell
yarn release
```

Create minor
```shell
yarn release:minor
```
